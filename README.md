# rain-history

Daily precipitation data extraction and storage for historical analysis.

## Purpose

- Extract daily precipitation records for a specific location from a public website
- Store data in a custom AWS database
- Manage all AWS infrastructure as code (Terraform) and CI/CD (CircleCI)
- Use TypeScript for the Lambda function with unit tests for the parser

## Tech Stack

- TypeScript
- AWS Lambda
- Terraform
- CircleCI
